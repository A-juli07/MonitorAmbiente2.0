Monitoreo de Temperatura, Humedad y Carbono

Objetivos:

  • Recolectar datos de temperatura, humedad y carbono en tiempo real utilizando el microcontrolador ESP32 y los sensores DHT11 y MQ-135.
  • Transmitir estos datos a la plataforma ThingSpeak para visualización.
